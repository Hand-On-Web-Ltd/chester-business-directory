# Chester Business Directory

A simple, open-source business directory for Chester, UK. Browse local businesses by category, search by name, and view details — all running in the browser with no backend.

Data is stored in a local JSON file so anyone can fork this and add their own listings.

## Features

- **Category filters** — Restaurants, cafes, shops, services, tech, and more
- **Search** — Find businesses by name or description
- **Detail view** — Click any listing for full info
- **Map placeholder** — Ready for Google Maps or Leaflet integration
- **Mobile friendly** — Works on any screen size

## Getting Started

1. Clone the repo
2. Open `index.html`
3. Browse Chester businesses

## Adding Businesses

Edit `data/businesses.json` and follow the existing format:

```json
{
  "name": "Business Name",
  "category": "Category",
  "description": "What they do",
  "address": "123 High Street, Chester",
  "phone": "01234 567890",
  "website": "https://example.com"
}
```
